Arduino REST API, Windows Serial, PHP7 Serial

> Home > Français > Ateliers > Arduino REST API, Windows Serial, PHP7 Serial

Durée : 4 ateliers ZOOM en soirée à 20h00 les mardis et jeudis 5, 7, 12, 14 mai 2020
Pré-requis :
  • Posséder une carte Arduino Uno ou éq. ou ESP32
  • Posséder un afficheur « I2C 2004 » ou similaire
  • Avoir installé l’environnement Arduino 1.8.12
1er atelier mardi 5 mai 2020 20h00: Présentation des ateliers
  • Accueil des participants
  • Vérification des environnements des participants
  • Démonstration succinte des ateliers
2e atelier jeudi 7 mais 2020 20h00: Implémentation d'un REST API
  • Compilation Arduino d'un programme REST API
  • Test à l'aide du moniteur intégré de l'IDE Arduino
  • Test avec PuTTY

3e atelier mardi 12 mai 2020 20h00: Programme Serial pour Windows
  • Installation du compilateur GCC pour Windows
  • Compilation Windows d'un programme Serial
  • Test de communication Arduino REST API depuis la ligne de commande

4e atelier jeudi 14 mais 2020 20h00: Programme Serial pout PHP7
  • Installation d'un serveur Apache PHP7
  • Mise en place d'un site PHP Serial
  • Test de communication Arduino REST API depuis une page PHP
