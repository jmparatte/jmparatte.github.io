Téléchargement de Python 3

Le téléchargement est effectué depuis la Page des téléchargements Python Releases for Windows:

  1. Identifier la section Stable Releases dans la colonne de gauche
  2. Cliquer le lien Download Windows installer (64-bit) au bas de la section

Installation de Python 3

L'installation démarre en mettant en exécution le programme téléchargé python-3.9.2-amd64.exe (ou une version plus récente, 3.9.2 au moment de la rédaction de cette page).

  1. Démarrer le programme python-3.9.2-amd64.exe:

    Remarques:

    1. Install Now installera Python 3 uniquement pour l'utilisateur en cours.
    2. Install laucher for all users (recommended) grisé ne sera pas installé.
    3. add Python 3 to PATH désélectionné ne sera pas inscrit.

    Les valeurs par défaut proposées par l'installateur sont recommandées car la confusion des versions Python 2 et 3 pourraient être une source de dérangements pour d'autres applications Python.

  2. Cliquer sur le bouton Install Now pour poursuivre automatiquement l'installation:

  3. Cliquer sur le bouton Close pour achever l'installation.

Création de la variable d'environnement PYTHON3_EXE

La variable d'environnement PYTHON3_EXE indique la version appropriée de Python à utiliser dans les scripts.

  1. Copier Python 3.9 (64-bit) dans la zone de recherche Windows Taper ici pour rechercher:

    (Écrire Python 3 dans la zone de recherche est suffisant.)

  2. Cliquer la commande Ouvrir l'emplacement du fichier:

  3. Afficher les propriétés de Python 3.9 (64-bit):

    (Par le menu de l'explorateur de fichiers ou par Alt-Enter.)

  4. Copier le chemin Démarrer dans et le coller dans un Explorateur de fichiers:

  5. Taper la commande cmd.exe /k "SET PYTHON3_EXE=%CD%\python.exe&SET PYTHON3_EXE" dans la zone d'adresse et exécuter:

  6. Une Invite de commandes s'ouvre et affiche le résultat de la commande:

  7. Taper la commande %PYTHON3_EXE% pour vérifier le démarrage de Python 3:

    Le programme Python démarre et affiche la version, actuellement 3.9.2.

  8. Taper Ctrl-Z Enter pour quitter Python.
  9. Taper la commande exit pour fermer l'invite de commandes.
